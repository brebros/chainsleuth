# ChainSleuth 🔍

**AI-powered rug pull detector — paste a contract, get the truth.**

## What is ChainSleuth?

ChainSleuth is an AI agent that analyzes crypto token contracts for rug pull indicators. Just paste a contract address and get an instant risk assessment.

## How it works

1. **Paste** a token contract address (EVM)
2. **Analyze** — the app fetches on-chain data: holder distribution, liquidity lock status, ownership, contract verification, mint authority
3. **Score** — AI generates a risk score (1-100) with plain-language explanation
4. **Decide** — clear green/yellow/red indicators so you know if it's safe

## Built on 0G

- **0G Compute** — AI inference for contract analysis & risk scoring
- **0G Storage** — scan history saved permanently
- **0G Chain** — on-chain verification of analysis results

## Why this matters

Rug pulls cost investors millions every year. Most victims don't know how to read smart contracts. ChainSleuth makes security accessible to everyone — no technical knowledge required.

## Tech Stack

- Frontend: React + TailwindCSS

- Storage: 0G Storage (decentralized storage)
- Chain: 0G Chain (on-chain verification)
